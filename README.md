# Chat

1. Clone project
2. ```yarn install```
3. Copy file ```.env.template``` rename it to ```.env```
4. Change all the environment variables
5. Run the database
```
docker-compose up -d
```
6. Ejecutar seed
```
http://localhost:3000/api/seed
```

7. Run the server develop mode: ```yarn start:dev```
